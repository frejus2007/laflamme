RÈGLES ET POINTS

📜 Introduction

Bienvenue dans le manuel officiel du clan La Flamme. Le système de points du clan est strict, automatique, et valide par l'administration.

Les points d'un joueur suite à un match sont calculés selon la base suivante :

Points du Match = Points de Rang + Bonus Intra + Bonus MVP + Bonus Admin

Note importante : Les KILLS sont enregistrés sur votre profil et aident au classement, mais ne donnent pas directement de points clan (hors départage ou rang interne).

🎮 MODE MJ (Multijoueur)

🔷 Principe général :

Points basés sur :

  • classement individuel (1 à 5)
  • résultat de l'équipe
  • performance (kills)
  • bonus/malus

👉 Score MAX = 33 points (hors bonus admin)

🔢 1. Barème des points :

Rang → (Équipe gagnante) / (Équipe perdante)

Rang Équipe gagnante Équipe perdante
1er +15 +10
2e +12 +8
3e +9 +6
4e +6 +3
5e +4 0

⚠️ 2. Malus performance :

  • 5e + défaite → 0 point
  • 4e + défaite → -1 point
  • 5e deux fois d'affilée (même session) → -2 points

🔫 3. Kills (bonus progressif)

  • 0 à 10 kills → +0,5 / kill
  • 11 à 20 → +0,3 / kill
  • 21+ → +0,1 / kill

🔒 Limites :

  • Cap kills = +6
  • Bonus :
    • 30+ kills → +1
    • 50+ kills → +2

👉 MAX kills = +8

⚠️ Les kills ne peuvent pas dépasser 50% des points de base

🏆 4. Bonus gameplay :

  • MVP → +5

👉 Inclus dans le plafond des 33

🚫 5. Bonus admin :

  • Bonus admin → +3 à +5

👉 ❗ NE COMPTE PAS dans les 33 points
👉 Peut faire dépasser le maximum

📊 Exemple MAX MJ (gameplay uniquement) :

  • 1er + victoire → 15
  • Kills max → +8
  • MVP → +5
  • Ajustement performance (kills élevés) → +5

➡️ TOTAL = 33 points

📊 Résumé MJ :

Score gameplay = Base + Kills + MVP − Malus

Score final = Score gameplay + Bonus admin

👉 Max gameplay = 33
👉 Bonus admin = au-dessus

🪂 MODE BR (Battle Royale)

🔷 Principe général :

Points basés sur :

  • classement de l'équipe
  • rang dans l'équipe
  • performance (kills)
  • bonus/malus

👉 Score MAX = 33 points (hors bonus admin)

🔢 1. Points équipe :

Points équipe =

arrondi( (Nombre de teams - Rang + 1) / Nombre de teams × 12 )

👉 MAX ≈ 12 points

🔢 2. Bonus intra-équipe :

  • 🔹 Solo :
    • Aucun
  • 🔹 Duo :
    • 1er → +3
    • 2e → 0
  • 🔹 Squad :
    • 1er → +4
    • 2e → +2
    • 3e → 0
    • 4e → -2

⚠️ 3. Malus performance :

  • Dernier 30% des équipes → -2
  • Dernier joueur + mauvaise team → -2

🔫 4. Kills (BR)

  • 1 kill = +0,7

🔒 Limites :

  • Cap kills = +6
  • Bonus :
    • 5+ kills → +1
    • 10+ kills → +2
    • 15+ kills → +3

👉 MAX kills = +9

⚠️ Ajustement :

  • Si équipe hors top 50% :
    • kills ÷ 2
    • cap réduit à +4

🏆 5. Bonus gameplay :

  • MVP :
    • Solo → 0
    • Duo → +4
    • Squad → +5

👉 Inclus dans le plafond des 33

🚫 6. Bonus admin :

  • Bonus admin → +3 à +5

👉 ❗ NE COMPTE PAS dans les 33 points

📊 Exemple MAX BR (gameplay uniquement) :

  • 1er équipe → 12
  • 1er squad → +4
  • Kills optimisés → +9
  • MVP → +5
  • Ajustement performance → +3

➡️ TOTAL = 33 points

📊 Résumé BR :

Score gameplay = Points équipe + Intra-team + Kills + MVP − Malus

Score final = Score gameplay + Bonus admin

👉 Max gameplay = 33
👉 Bonus admin = au-dessus

⚖️ ÉQUILIBRAGE FINAL

  • ✔ MJ max = 33 (gameplay)
  • ✔ BR max = 33 (gameplay)
  • ✔ Bonus admin séparé

👉 Système équilibré, compétitif et non abusif
👉 Les kills comptent sans dominer
👉 Le classement reste prioritaire
