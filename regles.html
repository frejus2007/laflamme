RÈGLES ET POINTS

📜 Introduction

Bienvenue dans le manuel officiel du clan La Flamme. Le système de points du clan est strict, automatique, et valide par l'administration.

Les points d'un joueur suite à un match sont calculés selon la base suivante :

Points du Match = Points de Rang + Bonus Intra + Bonus MVP + Bonus Admin

Note importante : Les KILLS sont enregistrés sur votre profil et aident au classement, mais ne donnent pas directement de points clan (hors départage ou rang interne).

🎮 MODE MJ (Multijoueur)

Principe général :

Les points sont attribués selon :

  • le rang individuel dans la partie (1 à 5)
  • et le résultat de l'équipe (victoire ou défaite)

Barème des points :

Rang Équipe gagnante Équipe perdante
1er +15 +10
2e +12 +8
3e +9 +6
4e +6 +4
5e +4 +2

Règles importantes :

  • L'équipe gagnante gagne plus que l'équipe perdante
  • Le classement individuel reste déterminant

🏆 Bonus :

  • MVP : +5 points
  • Bonus admin : +3 à +10 points

⚠️ Formation des équipes :

  • Les équipes sont formées aléatoirement
  • Si une équipe dépasse 5 joueurs :
    • Certains joueurs ne jouent pas une manche
    • Ces joueurs reçoivent automatiquement les points du 4e de leur équipe

🪂 MODE BR (Battle Royale)

Principe général :

Les points sont calculés selon :

  • le classement final de l'équipe
  • le classement du joueur dans son équipe

Calcul des points :

  • Points équipe =
    arrondi( (Nombre de teams - Rang équipe + 1) / Nombre de teams × 20 )

Bonus intra-équipe :

  • Solo : aucun
  • Duo :
    • 1er → +3
    • 2e → -1
  • Squad :
    • 1er → +5
    • 2e → +2
    • 3e → 0
    • 4e → -4

🏆 Bonus :

  • MVP :
    • Solo → 0
    • Duo → +5
    • Squad → +6
  • Bonus admin : +3 à +10 points

À savoir :

  • Les kills sont affichés mais ne comptent pas dans les points
  • Le classement d'équipe est le facteur principal
  • Le rang dans l'équipe permet de différencier les joueurs
