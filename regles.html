RÈGLES ET POINTS

📜 Introduction

Bienvenue dans le manuel officiel du clan La Flamme. Le système de points du clan est strict, automatique, et valide par l'administration.

Les points d'un joueur suite à un match sont calculés selon la base suivante :

Points du Match = Points de Rang + Bonus Intra + Bonus MVP + Bonus Admin

Note importante : Les KILLS sont enregistrés sur votre profil et aident au classement, mais ne donnent pas directement de points clan (hors départage ou rang interne).

🪂 Mode Battle Royale (BR)

En Battle Royale, les points de base dépendent du classement final de l'équipe comparé au nombre total d'équipes (slots).

Points de Base = Arrondi( (Nombre_Equipes − Rang_Equipe + 1) / Nombre_Equipes × 20 )

⭐ Bonus & Malus Spéciaux

  • MVP (Most Valuable Player) :
    • Mode MJ : +5 points
    • BR Duo (1er de l'équipe) : +5 points
    • BR Squad (1er de l'équipe) : +6 points

  • Bonus Admin : Variables. Les admins peuvent ajouter manuellement des points exceptionnels (ex: +2, +5, +10) via le formulaire.
  • Changement de Spécialité (MJ ↔ BR) : Pénalité stricte de -20 points. Maximum autorisé : 1 fois par saison.
